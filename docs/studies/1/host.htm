Study 1 Resizing IFrames

Adding new lines to a test document does not fire the window resize event. Instead, a ResizeObserver must be used.

The host document registers a load event handler on the iframe content window. When that event fires, a ResizeObserver can be set up to monitor the content document body (it must be the body element because the observer.observe method expects an element and does not accept a node.

The handler function is different for each IFRAME, because the handler function must be bound to that IFRAME – the "this" parameters is the only way to pass a reference to the IFRAME to the resize handler.

The handler does not use the size information passed in by the observer to resize the IFRAME, because the content document is significantly taller than the observed BODY element.
